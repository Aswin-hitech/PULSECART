**PULSECART**

Overview
This project implements a comprehensive Shopping Cart System in Java, simulating an online retail platform. It features user authentication with distinct roles (Customer, Admin), product catalog browsing, shopping cart management, and multiple payment options (UPI, Credit Card, Cash on Delivery). The system leverages Object-Oriented Programming principles for clear design, extensibility, and maintainability.

Features
Role-based user login and management

Product catalog with search and personalized recommendations

Shopping cart with stock validation and quantity control

Secure payment processing with exception handling

Order history tracking with persistent storage

File-based data management using CSV and serialization

Intuitive GUI built with Java Swing components

Getting Started
Clone the repository

Open in your preferred Java IDE (e.g., Eclipse, IntelliJ)

Ensure data/ folder with CSV files is present for products, users, and orders

Run the main class to start the application

Explore user login, shopping, and admin functionalities

Technologies Used
Java 11+

Java Swing for GUI

File I/O with CSV format

Java Serialization for object persistence

Custom exception handling
